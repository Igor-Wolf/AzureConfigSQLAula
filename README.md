# AzureConfigSQLAula
Resumo sobre a conteudo disponibilizado pela DIO referente a configuração de um banco de dados na Azure

Foi salientado que é necessario criar um servidor simbólico para alocar o sistema de sql, foi demosntrado como definir a forma de ligar o sql.
A partir das configurações a calculadora apresentou a previsão do valor de custo.
